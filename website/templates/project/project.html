Website development

This project involves development of a hospital management
website. The languages to be used are HTML,CSS, React and,
NodeJs. Any IDE eg, VScode can be used. MySQL Workbench
will be used as the database. Collaboration will be
virtual on GitHub.


Animation creation

This project involves creation of a school-based animation.
Alice3 will be the application used for creation. The characters
involved will be students, teachers, principal, nurse,
socialworkers. The aim of making the animation
is to use it to educate children on discipline and responsibility.


Debate Contest

This project involves planning and coordinating a debate
contest. Daily meetings will be happening from 1800hr to
1700hr in Microsoft Auditorium. During the meetings, the
debate motion will be selected. A certificate of participation
will be presented at the end of the debate.


Arduino Project

This project involves building an Automatic plant watering
system. Basically, a sensor will be used to detect when
the soil is dry and the Arduino will send signals that will
activate the water pump, irrigating the plants and when the
soil is moisturised the pump is switched off. An LED will also
be used to indicate when the pump is switched on/off.The project
will be virtually implemented on Tinkercard, then physically
implemented using the physical components.
